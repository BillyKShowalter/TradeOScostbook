# TradeOS UI Studio V2.0

Comprehensive web-based designer for the TradeOS ecosystem.

## 🚀 Features
- **3-Column Designer**: Visual palette on the left, canvas in the middle, and SwiftUI code export on the right.
- **Component Palette**: Rapidly design layouts using premium stat cards, charts, and cost items.
- **Live Data Integration**: Previews use real items from `Data/raw/items.json`.
- **Theme Studio**: Live update accent and secondary colors with instant SwiftUI theme export.

## 🛠️ How to Use
1. Open `tools/ui_studio/index.html` in your browser.
2. Select a view from the "Views" list (Command Center, Cost Library).
3. Tweak the **Design System** colors to see real-time updates.
4. Copy the generated **SwiftUI Code** from the right panel and paste it into your `Views/` directory in the Xcode project.
