# TradeOS Runtime - Integration & Future Roadmap

This document outlines the future development sprints for expanding the capabilities of the TradeOS Estimating App Runtime.

---

## 1. Feature Expansion Timeline

### Phase 1: Intake & Capture Expansion (Q4 2026)
* **Photo Estimating**: Deploy fine-tuned vision models capable of counting shingles, spotting fence deterioration levels, and measuring site layouts.
* **Drone Ingestion**: Ingest photogrammetry logs and orthomosaic maps to measure complex roof heights and slope angles automatically.
* **Blueprint Parsing**: Auto-takeoff module parsing large scale CAD/PDF plans to count electrical symbols and structural footings.

### Phase 2: Knowledge & Pricing Integrations (Q1 2027)
* **Permit Lookup**: Query county database APIs dynamically to retrieve local zoning and arborist permit rules.
* **Live Material Pricing**: Sync with Home Depot Pro, Lowe's, and Ferguson wholesale pricing APIs to refresh local material costs daily.
* **Building Code Awareness**: Cross-reference estimate items against local building code guidelines to identify structural reinforcement gaps.

### Phase 3: Project Planning & Operations (Q2 2027)
* **Predictive Labor**: Use historical crew timesheets to predict labor speeds based on local weather conditions.
* **Mobile AI Assistant**: Interactive voice estimating assistant for contractors walking site perimeters in real time.
