One-line: The TradeOS logotype in Space Grotesk with a copper "OS" — use in nav bars, hero lockups, footers, and anywhere the brand name is set as type.

```jsx
<Wordmark size="lg" tone="dark" />
```

Variants:
- `size`: `sm` 18px (nav) · `md` 24px · `lg` 32px · `xl` 46px (hero/display).
- `tone`: `dark` (bone text, on forge backgrounds) · `light` (forge-black text, on bone).
- The "OS" tspan is always copper — do not change it.
